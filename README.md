# projet-annuel
## Projet Revan
Pour télécharger tous les nodes modules
```
npm i
```
Ouvrir deux terminaux, dans l'un
```
cd .\client\src\
npm start
```
Et dans l'autre
```
cd .\server\
npm start
```

Vous avez le server et le react qui tournent, vous pouvez utilisez le projet en vous connectant avec votre adresse MetaMask
